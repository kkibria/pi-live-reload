# pi-live-reload

## install
Install node and npm first.
```bash
apt-get update
apt-get upgrade
apt autoremove
sudo apt-get install nodejs npm
```

Install ``pi-live-reload``
```bash
curl --proto '=https' --tlsv1.2 -sSf https://raw.githubusercontent.com/kkibria/pi-live-reload/master/install.sh | sh
```
